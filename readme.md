Данный код реализует программу "Текстовый редактор" на языке программирования Python. 

Проект сделан в рамках индивидуального проекта на курсе Проектный семинар “Python в науке о данных”.

Автор: Грошева Анастасия.
В этом редакторе реализованы функции: 
- копировать/вставить,
- изменение стиля, начертания шрифта
- изменение цвета шрифта
- изменение цвета фона
- сохранение, открытие и создание нового файла
- возможность напечатать файл
- возможность выбрать тему

Этот текстовый редактор можно использовать в повседневной жизни для создания заметок, работы и так далее.
